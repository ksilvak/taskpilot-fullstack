# TaskPilot
Simple task overview app.
Own account with task management. 
Admin can also manage other users.
Used for my testing. 
Still under development.

---

## Project Setup

### Requirements
- Node.js
- npm

## Run Frontend
In frontend folder run following commands: 
  - npm install
  - npm run dev


## Run Backend
In backend folder run following commands: 
  - npm install
  - npm run start
